🎯 About T Tasks Timer
  • Version 1.0.0
  • Issue Tracker GitHub

T Tasks Timer is a lightweight, privacy-focused task runner and sequence scheduler designed to automate your time-blocks.

How It Works

  • ⚙️ Create Tasks & Actions: Build custom tasks. Each task contains individual actions with precise durations, custom spoken texts, or background alerts.
  • 🗓️ Design Your Schedule: Mix, match, and loop! Sequence your tasks in the timeline in any order you want (e.g., Task 1 → Task 1 → Task 2).
  • ⚡ Run & Execute: Watch your custom sequence execute with live countdowns, background audio, and text-to-speech announcements.

This app is 100% Free and Open Source under the Apache 2.0 License. It runs entirely locally in your browser and uses your device's capabilities.
